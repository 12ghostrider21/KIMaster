Module: DraggableImage

Vue component for displaying and dragging an image across the screen. This component is used in an active game to show the still-active game stage.

Data

Name Type Default value Description
position Array.<number> [0,0] The current position of the image on the screen.
position.x number 0 The x-coordinate of the image.
position.y number 0 The y-coordinate of the image.
dragging boolean false Indicates if the image is currently being dragged.
offset Array.<number> [0,0] The offset of the cursor from the image's top-left corner during dragging.
offset.x number 0 The x-offset of the cursor.
offset.y number 0 The y-offset of the cursor.
