@startuml

title Spielstart - Interaktion im Spiel

actor User
participant Webseite

User -> Webseite: Beginnt das Spiel
activate Webseite
Webseite -> Webseite: Lädt das Spiel 
Webseite --> User: Zeigt das Spielfeld an
deactivate Webseite

User -> Webseite: Interagiert mit dem Spielfeld (macht Züge, Spieleabbruch)
activate Webseite
Webseite -> Webseite: Verarbeitet Benutzeraktionen
Webseite --> User: Aktualisiert das Spielfeld
deactivate Webseite

User -> Webseite: Nutzt Rückkehr-Button
activate Webseite
Webseite -> Webseite: Leitet den Benutzer zur Startseite weiter
Webseite --> User: Startseite mit Liste der Spiele anzeigen
deactivate Webseite
@enduml
