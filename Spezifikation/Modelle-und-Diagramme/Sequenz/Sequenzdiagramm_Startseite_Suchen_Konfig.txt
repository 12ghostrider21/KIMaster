@startuml

title Startseite - Spiele anzeigen, suchen und konfigurieren

actor User
participant Webseite

User -> Webseite: Besucht die Startseite
activate Webseite
Webseite -> Webseite: Zeigt Startseite an
Webseite --> User: Liste der Spiele anzeigen
deactivate Webseite

User -> Webseite: Nutzt die Suchleiste
activate Webseite
Webseite -> Webseite: Verarbeitet Suchanfrage
Webseite --> User: Zeigt Suchergebnisse an
deactivate Webseite

User -> Webseite: Wählt ein Spiel aus
activate Webseite
Webseite -> Webseite: Öffnet Konfigurationsfenster
Webseite --> User: Zeigt Konfigurationsfenster an
deactivate Webseite

User ->> Webseite: Wählt Spielmodus (z.B. gegen KI)
activate Webseite
User -> Webseite: Wählt Spieloptionen (z.B. Spielfeldgröße)
Webseite -> Webseite: Verarbeitet Konfiguration
Webseite --> User: Aktualisiert die Seite und zeigt Spiel an
deactivate Webseite
@enduml
